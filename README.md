# [EU Order Withdrawal Button for WooCommerce](https://vendidero.com/implementing-legally-compliant-eu-order-withdrawal-button-in-woocommerce)

### Add a EU-compliant order withdrawal button to your WooCommerce shop
This plugin helps to comply with the latest [EU directive 2023/2673](https://eur-lex.europa.eu/eli/dir/2023/2673/oj/eng) to offer
a way for your customers to withdraw from contracts.

- Withdrawal request form for guests and customers
- Manage withdrawals (confirm, reject) from your WooCommerce dashboard
- Send confirmation emails
- Allow customers and guests to select which items to withdraw (partial withdrawals)

Find out more about EU Order Withdrawal Plugin for WooCommerce on our [WordPress Plugin Repository Page](https://wordpress.org/plugins/eu-order-withdrawal-button-for-woocommerce/).
